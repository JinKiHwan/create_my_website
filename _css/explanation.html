CSS 강의 자료

CSS란 무엇인가?

CSS(Cascading Style Sheets)는 HTML 문서의 스타일(디자인)을 지정하는 언어입니다. 이를 통해 색상, 글꼴, 배경, 레이아웃 등을 조정할 수 있으며, 웹사이트의 일관된 디자인을 유지할 수 있습니다.

CSS 적용법

1. 인라인 적용법

<p style="color: blue; font-size: 18px;">이 문장은 파란색이며, 글자 크기는 18px입니다.</p>

2. HTML 내부에서 선언

<style>
p {
    color: red;
    font-size: 20px;
}
</style>

3. 외부 CSS 파일 링크

<link rel="stylesheet" type="text/css" href="styles.css">

CSS 선택자 사용법

1. 태그 선택자

h1 { color: blue; }

2. 클래스 선택자

.highlight { color: orange; font-weight: bold; }

3. 아이디 선택자

#unique { background-color: yellow; }

CSS 우선순위

우선순위 순서: !important > 인라인 스타일 > ID 선택자 > 클래스 선택자 > 태그 선택자

CSS 속성과 값

1. 색상 및 글꼴 관련 속성

color: red; font-size: 16px; font-weight: bold; text-align: center;

2. 박스 모델 속성

width: 100px; height: 50px; padding: 10px; margin: 20px; border: 1px solid black;

3. 배경 관련 속성

background-color: lightgray; background-image: url('image.jpg'); background-size: cover;
