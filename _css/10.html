🔍 Reset CSS란?

Reset CSS는 브라우저마다 다른 기본 스타일을 제거하여 스타일의 일관성을 유지하는 기법입니다.
브라우저 기본 스타일이 다르기 때문에 같은 코드를 작성해도 결과가 달라질 수 있습니다.

⚖️ Reset CSS 전후 비교

🚫 Reset CSS 미적용

이 단락은 기본 브라우저 스타일을 따릅니다.

  • 기본 마진 & 패딩 있음
  • 브라우저마다 다르게 보일 수 있음

✅ Reset CSS 적용

이 단락은 Reset CSS를 적용한 결과입니다.

  • 모든 마진 & 패딩 제거
  • 일관된 스타일 유지

📌 Reset CSS 예제

다음은 가장 기본적인 Reset CSS 코드입니다:

/* Reset CSS */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
li,ol{
    list-style:none;
}
