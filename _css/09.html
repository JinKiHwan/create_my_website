CSS에서 폰트 적용 방법

1. Link 방식

Google Fonts 등의 외부 폰트를 <link> 태그로 불러오는 방식

장점: 간편하고 CDN을 활용하여 로딩 속도가 빠름

단점: 인터넷이 필요함

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

body {
  font-family: 'Roboto', sans-serif;
}
이것은 Roboto 폰트입니다.

2. Import 방식

@import를 사용하여 CSS 파일 내에서 폰트를 불러오는 방식

장점: CSS 파일에서 관리 가능

단점: 추가적인 HTTP 요청으로 인해 로딩 속도가 느릴 수 있음

@import url('https://fonts.googleapis.com/css2?family=Open+Sans:wght@400;700&display=swap');

body {
  font-family: 'Open Sans', sans-serif;
}
이것은 Open Sans 폰트입니다.

3. WOFF 파일 방식

로컬 폰트 파일(.woff, .woff2)을 직접 적용하는 방식

장점: 인터넷 연결 없이 사용 가능

단점: 서버에 폰트 파일을 직접 업로드해야 함

@font-face {
  font-family: 'CustomFont';
  src: url('fonts/CustomFont.woff2') format('woff2'),
       url('fonts/CustomFont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

body {
  font-family: 'CustomFont', sans-serif;
}
이것은 CustomFont 폰트입니다.
