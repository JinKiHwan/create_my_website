👩‍🏫 JavaScript 기초: 변수와 함수

1. 변수란 무엇인가요?

변수는 정보를 담는 상자와 같아요. 이름을 붙여서 나중에 다시 꺼내 쓸 수 있어요.

let name = "Kihwan";
let age = 20;

이렇게 하면 name에는 "Kihwan"라는 이름이, age에는 20이라는 숫자가 저장돼요.

2. 함수란 무엇인가요?

함수는 어떤 일을 묶어서 저장해 둔 것이에요. 필요할 때 이름을 부르면 실행돼요.

function sayHello() {
  alert("안녕하세요!");
}

이렇게 만든 함수는 sayHello()라고 부르면 작동해요.
