HTML 태그별 기본 디스플레이 속성

✅ 1. 인라인 (inline) 태그

특징: 한 줄을 차지하지 않으며, 내용 크기만큼만 공간을 차지함.

태그 설명
<a> 링크(anchor)
<span> 의미 없는 인라인 컨테이너
<strong> 굵게 표시
<em> 기울임(이탤릭, 강조)
<code> 코드 블록(소스 코드 강조)
<b> 굵게 표시 (의미 없음)
<i> 기울임 (의미 없음)
<u> 밑줄
<small> 작은 글씨
<mark> 형광펜 효과 (텍스트 강조)
<sub> 아래 첨자 (subscript)
<sup> 위 첨자 (superscript)
<q> 인라인 짧은 인용 (quote)

이 문장에서 이 부분은 인라인입니다.

이 링크도 인라인

✅ 2. 인라인-블록 (inline-block) 태그

특징: 인라인처럼 한 줄에 배치되지만, 블록처럼 width, height 지정 가능.

태그 설명
<img> 이미지(image)
<input> 입력 필드(input)
<button> 버튼(button)
<select> 드롭다운 리스트(select)
<textarea> 여러 줄 입력 필드
<label> 입력 요소와 연결(label)

✅ 3. 블록 (block) 태그

특징: 한 줄을 전체 차지하며 자동으로 줄 바꿈이 됨.

태그 설명
<div> 의미 없는 블록 컨테이너
<p> 단락(Paragraph)
<h1> ~ <h6> 제목(Heading) 태그
<ul>, <ol>, <li> 목록(List) 태그
<table> 표(Table) 태그
<form> 폼(Form) 태그
<section> 문서 구획
<article> 독립적 콘텐츠 블록
<aside> 사이드 콘텐츠
<footer> 푸터(Footer)
<header> 헤더(Header)
<nav> 내비게이션(Navigation)

제목입니다.

이것은 단락입니다.
